# 注册账号

## 一、账号介绍
> 管理员账号专为官方网站的后台管理作用，不适用于客户端登录。

> 为确保您的设备安全性与个性化管理，我们设置的客户端登录流程要求您在官方网站的后台管理系统中进行创建。每个设备需有一个独立的客户端账号，体验专属的服务。
