#### 客户端支持哪些平台?

> 目前支持Vista，Win7，Win10的32位及64位系统；WinXP部分支持，可能会有兼容性问题


### 我所有设备上的程序已经显示运行成功，但是我Ping不通对方，为什么？

> 检查自己的机器是否开启了防火墙，可以先用机器的原有IP Ping一下，看看通不通；
